# javaTasks

A collection of my Java exercises completed through [Exercism](https://exercism.org/) with automatic GitHub syncing enabled.  
Each folder contains an individual task with my solution.

## Structure
- Each directory represents one Exercism Java exercise  
- Solutions follow clean, readable Java practices  
- Tasks are synced automatically from Exercism

## Purpose
This repository serves as:
- A personal archive of solved Java challenges  
- A way to track my progress while learning Java  
- A reference for future practice and improvement
